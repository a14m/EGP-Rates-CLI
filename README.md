# EGPRatesCLI

CLI for showing and updating EGP exchange rate on terminal screen (every X interval)

Built using [EGP-Rates](https://github.com/mad-raz/EGP-Rates)

## Installation

    $ gem install EGP_Rates_CLI

## Usage

    $ egp
